# superbvue-docs

## Development

This site is powered by [VuePress](https://vuepress.vuejs.org/).

``` bash
cd docs 
yarn install
yarn dev
```
Or
``` bash
cd docs
npm install
npm run dev
```

This doc is under WIP. You can [open an issue](https://github.com/superbvue/superbvue-docs/issues/new) for bugs or feature requests.
